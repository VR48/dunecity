# 2026-07-05 Android Debug Checkpoint

This checkpoint makes Dune2R playable on a tested Android 13 device through the
SDL2 Android activity path.

Highlights:

- Builds the native game as Android `libmain.so`.
- Packages a debug APK through `scripts/package-android-apk.ps1`.
- Locks the Android activity to landscape for readable RTS controls.
- Adds Android app-storage and payload lookup paths.
- Stages the existing desktop icon as the Android launcher icon.
- Maps normal touch to the existing mouse-driven UI.
- Maps Android Back to Escape for menus and the map editor.
- Adds a two-finger right-click gesture: hold one finger on the target and tap a
  second finger.
- Adds Dune2R graphics override scaffolding for compact and HD objpic assets.
