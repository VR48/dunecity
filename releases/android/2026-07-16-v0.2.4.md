# Dune Legacy Android 0.2.4

Android ARM64 build based on DuneCity/Tornie 1.0.520, with the Dune2R mod
scaffold and Android runtime adaptations layered on top.

## Included

- Tornie Mod 1.0.520 source, campaigns, units, structures, and assets
- Dune City and vanilla Dune Legacy modes
- Selectable Dune2R graphics mod scaffold
- Landscape display and touch input
- Android Back mapped to Escape
- Two-finger right-click gesture
- On-screen keyboard support for save names and other text fields
- Android-local save loading and campaign save compatibility fixes
