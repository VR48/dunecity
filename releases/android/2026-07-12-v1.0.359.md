# Dune Legacy Android 1.0.359

Android ARM64 build based on Stefan's DuneCity/Tornie `v1.0.359` release,
with the Dune2R mod scaffold and Android runtime adaptations layered on top.

## Included

- Current DuneCity and Tornie game, campaign, mod, and data files
- Dune2R selectable mod scaffold
- Landscape presentation and correctly aligned single-touch input
- Android Back button mapped to Escape
- Two-finger tap mapped to right-click
- Versioned payload refresh for future upstream updates

## Compatibility

- Architecture: `arm64-v8a`
- Minimum Android API: 23 (Android 6.0)
- Target Android API: 34
- Tested on Android 13 with a Ulefone Armor 21
