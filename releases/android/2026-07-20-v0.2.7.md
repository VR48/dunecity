# Dune Legacy Android 0.2.7

Android ARM64 build based on DuneCity 1.0.523, with selectable DuneCity,
Tornie, and Dune2R modes.

## Included

- Generic mod-scoped ninth-house support and custom-house presentation hooks
- Tornie 1.0.523 campaigns, Tharpique content, graphics, sounds, and balance data
- Correct Tharpique special units, IX discovery, and worm-sign warning behavior
- Updated Advanced Windtrap and Scoutpost animation behavior
- Existing DuneCity simulation persistence and gameplay corrections
- Landscape touch input, two-finger right click, Android Back as Escape, and
  on-screen keyboard support
- Android-local campaign and save loading
- HTTPS multiplayer listing and cross-platform network support

## Compatibility

- ARM64 Android 6.0 or newer
- Existing Android application data and saves are preserved during upgrade.
- Multiplayer peers must use the current network protocol.
