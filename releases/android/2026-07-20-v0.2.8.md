# Dune Legacy Android 0.2.8

Android ARM64 hotfix based on DuneCity 1.0.524.

## Fixed

- Install the complete bundled Tornie payload, including all campaigns,
  Tharpique custom-house registration, Mentat declarations, graphics, sounds,
  and integrity metadata.
- Preserve Tornie's complete `mod.ini` during initial installation and upgrade.
- Detect and repair incomplete or outdated Tornie payloads using required-file
  checks and `checksums.sha256`.

## Compatibility

- ARM64 Android 6.0 or newer
- Existing Android application data and saves are preserved during upgrade.
- Save format and multiplayer protocol are unchanged from Android 0.2.7.
