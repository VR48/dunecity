# Dune Legacy Android 0.2.0

Android ARM64 update based on DuneCity/Tornie `1.0.359`, with the Dune2R mod
scaffold and Android runtime adaptations layered on top.

## Fixed

- Editable fields now open the Android soft keyboard when focused.
- Save Game names can be entered directly on mobile.
- Keyboard input also works consistently for player names, chat, map metadata,
  network settings, and other shared text fields.

## Android Versioning

- Android version name: `0.2.0`
- Android version code: `1000360`
- Bundled DuneCity version: `1.0.359`

Android package versions are now maintained independently from DuneCity game
versions.

## Compatibility

- Architecture: `arm64-v8a`
- Minimum Android API: 23 (Android 6.0)
- Target Android API: 34
- Tested on Android 13 with a Ulefone Armor 21
