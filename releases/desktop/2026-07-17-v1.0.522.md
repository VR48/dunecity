# DuneCity 1.0.522 Desktop

This release combines the Tornie 1.0.522 content update with restored DuneCity
simulation persistence, gameplay fixes, complete selectable-mod packaging, and
cross-platform multiplayer networking.

## Included

- Tornie 1.0.522 campaigns and balance data
- DuneCity R/C/I, city budget, and city-credit save persistence
- Classic production timing for concrete slabs
- Road replacement over concrete and concrete replacement over roads
- Correct infantry rifle sound when rocket infantry use their close-range gun
- Complete selectable mod packaging for DuneCity, Tornie, and Dune2R
- Active-mod loose graphics loading with safe fallback sprites
- HTTPS metaserver support for Android clients

## Verification

- Windows Release build and packaged runtime launch
- DuneCity, Tornie, and multiplayer desktop smoke tests
- Catch2 regression suite
- Android ARM64 cross-platform multiplayer smoke test
