# DuneCity 1.0.523 Desktop

This release adds generic ninth-house engine support and the Tornie 1.0.523
content milestone while preserving the existing behavior of vanilla Dune
Legacy, DuneCity, and Dune2R.

## Included

- Generic, mod-scoped ninth-house registration with legacy eight-house support
- Generic custom-house Mentat, herald, palette, voice, and presentation hooks
- Tornie 1.0.523 campaigns, Tharpique house content, graphics, sounds, and data
- Correct Tharpique special-unit selection and IX availability filtering
- Updated Advanced Windtrap graphics, preview, recoloring, and animation
- Restored Tornie structure-flag brightness and standard Scoutpost animation
- Correct one-time worm-sign warning behavior for the ninth house
- Versioned save and multiplayer compatibility for synchronized ninth-house state

## Compatibility

- Existing eight-house saves remain loadable.
- Multiplayer peers must use the current network protocol.
- Vanilla, DuneCity, Dune2R, and Tornie remain separately selectable modes.
