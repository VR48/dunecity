# DuneCity 1.0.520 Desktop

This release integrates Tornie Mod 1.0.520 with the transferred DuneCity root
repository while preserving Dune City, vanilla Dune Legacy, Android support,
and the selectable Dune2R graphics mod scaffold.

## Included

- Tornie 1.0.520 units, structures, campaigns, palettes, voices, and tools
- Random enabled special-vehicle selection using the synchronized game RNG
- Dune2R remains an independent selectable mod
- Tornie-only assets are gated by the active Tornie mod
- Multi-house Dune City RCI and budget state remains save-compatible
- Loaded-map and city-simulation exception guards remain in place
- Null-safe destroyed-target handling remains in place
